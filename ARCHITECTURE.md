## 三、组件说明

### 3.1 Controller 层

| 组件 | 功能 | 端点 |
|------|------|------|
| ImportController | 数据导入 | POST /api/import/{dirPath} |
| SearchController | 向量检索 | GET /api/search?query=xxx |
| RagController | RAG问答 | POST /api/rag/ask, POST /api/rag/ask-with-sources |

### 3.2 Service 层

| 组件 | 功能 |
|------|------|
| DataImportService | 文件读取、文本分块、向量化存储 |
| RagService | RAG流程编排、缓存集成、LLM调用 |
| QdrantService | 向量数据库操作封装 |
| FileReaderService | 文件读取工具 |

### 3.3 Utility 层

| 组件 | 功能 |
|------|------|
| EmbeddingCache | 向量缓存、减少API调用 |

### 3.4 外部依赖

| 服务 | 用途 | 配置 |
|------|------|------|
| Qdrant | 向量数据库 | localhost:6334 |
| 智谱AI | Embedding + Chat | api-key + base-url |
